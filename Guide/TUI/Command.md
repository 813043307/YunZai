## tmux 终端会话管理

| 名称         | 用法        |
| ------------ | ----------- |
| 离开窗口     | Ctrl+B D    |
| 切换窗口     | Ctrl+B W    |
| 新建窗口     | Ctrl+B C    |
| 分屏         | Ctrl+B -/\_ |
| 切换鼠标模式 | Ctrl+B M    |
| 切换复制模式 | Ctrl+B ↲    |
| 杀死窗口     | Ctrl+B X Y  |

## micro 文本编辑

| 名称       | 用法     |
| ---------- | -------- |
| 保存退出   | Ctrl+Q Y |
| 不保存退出 | Ctrl+Q N |
